he one big problem with CSS opacity is that it’s inherited by the contents of the element you’re applying it to, as well as the background. So if you look closely at Figure 4-23, you’ll see that the text on the page is showing through the alert text as well. This doesn’t matter if you’re using a very high opacity in combination with high-contrast text. However, with lower opacities, the content of your box can start to get unreadable.
